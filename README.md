# Adota-Pet 🐾

Plataforma centralizadora para apoio a ONGs animais, geolocalização de serviços e triagem preliminar com IA.

## 🚀 Tecnologias
- **Frontend:** React.js
- **Backend:** Node.js / Express
- **Banco de Dados:** Relacional (PostgreSQL/MySQL)
- **Integrações:** OpenAI API (Triagem) e Google Maps API (LBS)

## 🛠️ Como rodar o projeto
1. Clone o repositório.
2. No `/server`, execute `npm install` e `npm start`.
3. No `/client`, execute `npm install` e `npm start`.
